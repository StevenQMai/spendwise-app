# Images Directory

This directory contains all images, screenshots, and diagrams for the SpendWise showcase website.

## Required Images

### Screenshots
Add screenshots of your application here:
- `dashboard.png` - Dashboard screen
- `expense-log.png` - Expense Log screen
- `budgets.png` - Budget Management screen
- `savings-circles.png` - Savings Circles screen
- `chatbot.png` - AI Chatbot interface

### Diagrams
Add UML and architecture diagrams:
- `class-diagram.png` - Design Class Diagram
- `sequence-diagram.png` - Sequence Diagrams
- `component-diagram.png` - Component Diagram

### Team Photos
Add team member profile photos:
- `avatar-[name].png` - Individual team member photos (recommended: 200x200px or larger, square)

### Placeholder Images
If you need placeholder images for testing, you can use:
- Online placeholder services (e.g., placeholder.com, via.placeholder.com)
- Or create simple colored rectangles as placeholders

## Image Guidelines

- **Format**: PNG or JPG
- **Optimization**: Compress images for web to reduce load times
- **Naming**: Use descriptive, lowercase names with hyphens (e.g., `dashboard-overview.png`)
- **Size**: Keep file sizes reasonable (< 500KB per image when possible)
- **Dimensions**: Screenshots should be at least 1200px wide for good quality

## Usage

Images are referenced in the markdown files using relative paths:
```markdown
![Alt text](/assets/images/dashboard.png)
```
