## How to Update This Page

1. **Add Team Member Information**
   - Replace placeholder names, roles, and contributions
   - Add actual profile photos to `assets/images/` directory
   - Update GitHub and LinkedIn links

2. **Add Profile Photos**
   - Save team member photos as `avatar-[name].png` or similar
   - Place in `assets/images/` directory
   - Update image `src` attributes in team member cards
   - Recommended size: 200x200px or larger (square aspect ratio)

3. **Customize Contributions**
   - List specific features or modules each person worked on
   - Highlight unique contributions
   - Include links to GitHub profiles or portfolios

4. **Highlight Website Contributors**
   - Specifically mention team members who worked on the GitHub Pages deployment
   - Detail their contributions to the showcase website
